## рҹ“– Uso

### Desarrollo Local
```bash

npm run build        # Build de producciГіn

npm run audit:accessibility  # Solo auditorГӯa de accesibilidad
```

### Scripts Disponibles
- `dev` - Inicia el servidor de desarrollo con hot reload
- `build` - Genera build optimizado para producciГіn
- `lint` - Ejecuta anГЎlisis de cГіdigo estГЎtico
- `preview` - Vista previa del build de producciГіn
- `audit` - Ejecuta auditorГӯa completa con Lighthouse
- `audit:accessibility` - AuditorГӯa solo de accesibilidad
